## Redidit

### Installation
$ git clone https://github.com/Limu123/redidit_ng_2.git <br>
$ cd redidit_ng_2/ <br>
$ (sudo) npm install <br>
$ grunt serve <br>
--> do not 'bower install'

### Test
$ karma test

### Build
$ grunt build

### Database
https://redidit.firebaseio.com/
